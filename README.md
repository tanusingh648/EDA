# EDA# Zomato Restaurant Data Analysis using Seaborn

## Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Zomato Restaurants Dataset using **Python**, **Pandas**, **Seaborn**, and **Matplotlib**. The objective is to understand restaurant distribution, identify trends, and visualize important insights from the dataset.

## Dataset

* **Dataset:** Zomato Restaurants Data
* **Source:** Kaggle
* **Link:** https://www.kaggle.com/datasets/shrutimehta/zomato-restaurants-data

## Technologies Used

* Python
* Pandas
* Seaborn
* Matplotlib
* KaggleHub

## Project Features

* Load the Zomato dataset from Kaggle.
* Perform basic data exploration.
* Remove duplicate records.
* Analyze restaurant distribution by country.
* Create clear and informative visualizations using Seaborn.

## Visualizations

The project includes visualizations such as:

* Restaurant Count by Country
* Distribution Plots
* Count Plots
* Bar Charts
* Box Plots
* Heatmaps (if applicable)

## How to Run

1. Clone the repository.
2. Install the required libraries using:

```
pip install -r requirements.txt
```

3. Open the notebook in Jupyter Notebook or Google Colab.
4. Run all cells to reproduce the analysis and visualizations.

## Key Insights

1. The dataset contains restaurant information from multiple countries.
2. Duplicate records are removed before analysis.
3. Restaurant distribution varies significantly across countries.
4. Seaborn visualizations help identify trends and patterns quickly.
5. The dataset can be extended for further analyses such as ratings, cuisines, pricing, and online delivery.
